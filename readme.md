# 🍃 CalcuLab — Your Perfect Hydroponics Setup Buddy

CalcuLab is a premium, minimalist, and highly interactive multi-page web application designed to eliminate the financial surprise and complexity of setting up a residential hydroponic farm. Built with a tranquil, cyberpunk-eco aesthetic, it empowers aspiring home growers to calculate exact configuration costs, explore dynamic hardware catalogs, and map out a clean path to their first harvest.

---

## 🚀 Live Demo & Key Pages

The ecosystem consists of four core modules engineered to deliver a seamless, native-feeling user experience:

* **`index.html` (Home Hub):** Introduces the core philosophy, features an automated, infinitely loop-scrolling testimonial track, and guides users directly into deep calculation zones.
* **`calcu.html` (Quantum Calculator Portal):** An immersive configuration sandbox where users select equipment nodes, route them down an active animated conveyor assembly array, and initiate a 3-second critical overload vortex sequence to instantly decipher real-time market balances.
* **`store.html` (Hardware Deck):** An integrated marketplace catalog sorted clearly by vital structural baselines and optional biotic optimization modules.
* **`guide.html` (Starter Guide Blueprint):** A simple, calm, three-phase onboarding roadmap translating spatial analysis, water circulation metrics, and chemical balances for complete beginners.

---

## ✨ Features

* **Premium Cyberpunk-Eco Theme:** Designed utilizing a glassmorphic color palette featuring deep midnight greens (`#0A140E`), vibrant neon leaf accents (`#39FF14`), and crisp white typography for high trust and luxury readability.
* **Indian Rupee (₹) Native Pricing Calibration:** All pricing catalogs, data sheets, and script evaluation total engines are completely hardcoded and formatted directly in Indian Rupees (`₹`), complete with localized thousands-separator logic (`toLocaleString('en-IN')`).
* **State-Enforced Input Security Controls:** The calculator core portal locks down interaction mechanics using real-time variable switches—rejecting double-trigger actions during countdowns or blocking vortex calculations when the active stream is empty.
* **Fluid Navigation Route Transitions:** Custom shared fade-in and upward slide slide-out page routers intercept standard browser jumps to completely eliminate heavy, distracting white screen flashes.
* **Fully Phone & Mobile Responsive:** Structured from the ground up using mobile-first CSS breakpoints, accommodating text tracking layouts, fluid button sizing grids, and auto-collapsing navigational elements comfortably on any smartphone screen orientation.

---

## 🛠️ Tech Stack & Architecture

The project is built explicitly with clean, vanilla web technologies to keep performance rapid, decoupled, and straightforward to maintain:

* **HTML5:** Semantic, clean markup layout architecture with validated tag structures.
* **CSS3:** Multi-page modular styling engine powered by CSS Variables, fluid flexbox tracks, responsive layout grids, and optimized canvas keyframe animations.
* **Vanilla JavaScript (ES6+):** Dynamic catalog population engine, conveyor timing offsets, mathematical accumulator arrays, and event handlers.
